# 🐾 RescuePaw Labs

Proyecto desarrollado para el hackathon **zkSYS**. 

## 🏗️ Estructura del Proyecto
- **smart-contracts/**: Contratos inteligentes en Solidity (Hardhat).

- **agente-autonomo/**: Lógica del agente para la gestión de rescates.
